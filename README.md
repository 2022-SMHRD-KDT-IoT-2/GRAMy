### Back-End

- 스네이크 케이스 (snake_case)
- 모든 단어를 소문자로 표기하고, 단어를 언더바(\_) 로 연결하는 방법

### Front-End

- 카멜 케이스 (camelCase)
- 첫단어는 소문자로 표기하지만, 이어지는 단어의 첫글자는 대문자로 표기하는 방법
